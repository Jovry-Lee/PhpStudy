## 杂项函数
### 1. unpack 从二进制字符串解包
#### 1.1 说明
格式:
```
unpack ( string $format , string $data [, int $offset = 0 ] ) : array
```
从二进制字符串解包, 通常打包的参数与解包的参数一致.

#### 1.2 参数
- format 格式,参见pack()函数.
- data :待解包的数据.
- offset: 解包偏移量

#### 1.3 返回值
返回二进制数据解包后的元素数组.

##### 1.4 示例
###### 1.4.1 解包L
```
<?php

$string = pack('L4', 1134242, 22423421, 142423423, 14234234);
var_dump(unpack('L4', $string));
```
测试结果:
```
array(4) {
  [1] =>
  int(1134242)
  [2] =>
  int(22423421)
  [3] =>
  int(142423423)
  [4] =>
  int(14234234)
}
```

###### 1.4.2 解包后指定key
```
$string = pack('L4', 1134242, 22423421, 142423423, 14234234);
var_dump(unpack('Ll1/Ll2/Ll3/Ll4', $string)); //可以指定key，用/分割
```
测试结果:
```
array(4) {
  'l1' =>
  int(1134242)
  'l2' =>
  int(22423421)
  'l3' =>
  int(142423423)
  'l4' =>
  int(14234234)
}
```
